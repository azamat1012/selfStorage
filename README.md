# selfStorage

## Запуск:

### 1

```bash
  cd selfStorage
```

### 2

```bash
  pip install -r requirements.txt
```

### 3

```bash
  python manage.py makemigrations
  python manage.py migrate
```

### 4 - Создать .env файл

### 5

```bash
  TG_BOT_TOKEN=....
  SECRET_KEY=....
```


### 6

```bash
  cd backend
```

### 7

```bash
  python main.py
```
